安装React脚手架：
1.全局安装 npm install -g create-react-app //不需要
2.创建项目 npx create-react-app <project-directory> //直接使用npx create-react-app创建项目即可
3.进入项目文件夹 cd <project-directory>
4.启动项目 npm start
5. ctrl+c stop running
